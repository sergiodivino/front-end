-Cria um novo array apenas com os itens que passaram no filtro
-Não modifica o array original
-----------------------------SINTAXE---------------------------------
array.filter(callback, thisArgs)
dentro de callback(item, index, array)
-CALLBACK- função a ser executada em cada elemento
obs: normalement é uma comparação
obs: callback pode passar apena um parâmetro
-thisArgs(opcional)pouo usado no dia a dia
obs: utilizado para variar a qual objeto a função de refere, 
se opere em objeto x ou y

--------------------EXEMPLO COM FILTER--------------------------------
const frutas = ['maçã fuji','maçã verde','laranja','abacaxi'];
frutas.filter((fruta)=> fruta.includes('maçã'))
// retorno: ['maçã fuji', 'maçã verde'];
